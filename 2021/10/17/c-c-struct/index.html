C/C++ struct

  • 自訂不同資料型態串在一起

    • 在C語言中，定義一個結構體型別要用typedef :
    • 在 C中，struct不能包含函式
    • 在C++中，對struct進行了擴充套件，可以包含函式。

  • struct 可以內嵌在結構內的結構

    • 結構內不能嵌入同一個結構，也就是結構不能遞迴宣告

  • 儲存結構的陣列

  • 存取結構指標的屬性

    • 存取結構指標內的屬性有兩種方式：
      • 解指標 (dereference)
      • 使用 -> (箭號)
    • 第二種方式在語法上比較簡潔，故較受歡迎，可以當做一種語法糖
