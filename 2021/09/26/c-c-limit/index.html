C/C++ limits

  • 在 c 中是 <limits.h>
  • 在 c++ 中是 <climits>
    :::info
    1、32位编译系统中:
    long 占4字节 int 占4字节 long int 占4字节
    2、64位编译系统中:
    long 占8字节 int 占4字节 long int 占8字节
    3、32位和64位编译系统中
    long long int 在32位和64位编译系统中，都占8字节
