Set Github SSH

gen ssh key

set ssh on github or gitlab

github

  • go to github setting
  • click SSH and GPG keys on the left bar
  • click new ssh key
  • paste the output of ~/.ssh/id_rsa.pub on it

gitlab

  • go to gitlab setting
  • click SSH keys on the left bar
  • click new ssh key
  • paste the output of ~/.ssh/id_rsa.pub on it
